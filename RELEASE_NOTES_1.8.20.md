# Folder Notes 中文版 v1.8.20

首个可发布版本。

## 亮点

- 完成主插件界面中文化
- 完成 `Folder Overview` 相关界面中文化
- 将子模块整理为普通源码目录，仓库可独立发布
- 更新发布清单、README 和 GitHub Actions

## 安装

下载并放入 Obsidian 插件目录：

- `main.js`
- `manifest.json`
- `styles.css`

插件目录建议为：

`Vault/.obsidian/plugins/folder-notes-zh/`

## 说明

本项目基于原作者 Lost Paul 的 `obsidian-folder-notes` 进行汉化和发布整理。
